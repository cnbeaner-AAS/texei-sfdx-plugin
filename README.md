## Install Plugin

### Install as plugin

Install plugin: `sf plugins install texei-sfdx-plugin`

### Install from source

Install the SF CLI.

Clone the repository: `git clone https://github.com/texei/texei-sfdx-plugin.git`

Install npm modules: `npm install`

Link the plugin: `sf plugins link ./`
